将一个函数作为另一个函数的参数称之为嵌套函数。Excel 中最多可以嵌套 64 层函数。

以下公式根据学生的得分区间给予一个综合评分，其中使用到了嵌套的 IF 函数。
<!-- more -->
```
=IF(B2>=90,"优",IF(B2>=75,"良",IF(B2>=60,"及格","差")))
```

上述公式中总计嵌套了 3 层 IF 函数。嵌套的部分作为 IF 函数判断错误部分进行二次、三次判断。将学生分数以60分、75分和90分分为三个档次进行评分。
